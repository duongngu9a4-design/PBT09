Giới Thiệu

Repository này được xây dựng để thực hiện Phiếu Bài Tập 09 với chủ đề DOM Manipulation & Events.

Bài tập tập trung vào việc tìm hiểu cách JavaScript tương tác với cấu trúc HTML thông qua DOM (Document Object Model), đồng thời xử lý các sự kiện phát sinh từ người dùng nhằm tạo ra các trang web có tính tương tác cao.

---

Mục Tiêu Bài Tập

- Hiểu cấu trúc và vai trò của DOM trong phát triển Web.
- Thực hành truy xuất và thao tác các phần tử HTML bằng JavaScript.
- Xử lý các sự kiện (Events) từ người dùng.
- Xây dựng giao diện tương tác động.
- Nâng cao kỹ năng lập trình Front-end bằng JavaScript thuần (Vanilla JavaScript).

Nội Dung Thực Hiện

DOM Manipulation

Các thao tác chính:

- Truy xuất phần tử bằng:
  
  - "getElementById()"
  - "getElementsByClassName()"
  - "getElementsByTagName()"
  - "querySelector()"
  - "querySelectorAll()"

- Thay đổi nội dung:
  
  - "innerHTML"
  - "textContent"

- Thay đổi thuộc tính:
  
  - "setAttribute()"
  - "removeAttribute()"

- Thao tác CSS:
  
  - "classList.add()"
  - "classList.remove()"
  - "classList.toggle()"

- Tạo và xóa phần tử:
  
  - "createElement()"
  - "appendChild()"
  - "remove()"

Event Handling

Các sự kiện được sử dụng:

Mouse Events

- click
- dblclick
- mouseover
- mouseout

Keyboard Events

- keydown
- keyup

Form Events

- submit
- change
- input
- focus
- blur

Window Events

- load

- Xử lý tương tác người dùng thông qua Events.
- Kiểm tra và phản hồi dữ liệu nhập từ người dùng.
